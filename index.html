Versioning control system

3 type of version control system

  1. Local Version Control
  2. Centralized Version Control
  3. Distributed Version Control

Some basic git commands are

  1. git add : take a snapshot of all content in the directory. stored in temporary staging area called index
  2. git commit : permenentity store the content of index to the repository.
  3. git diff : show the changes that we made but not yet added to the index
  4. git diff --cached : it show what about to be committed
  5. git status : get a bref summary
  6. git log : show the history of your changes
  7. git branch branch-name :create an new branch with name branch-name
  8. git branch -d branch-name :delete branch
  9. git branch -D branch-name :Force delete
  10. git branch -c branch-name :create new brnch and switch it to the new branch
  11. git switch :used to switch branch
  12. git checkout :switch branches and used to restore working tree files
